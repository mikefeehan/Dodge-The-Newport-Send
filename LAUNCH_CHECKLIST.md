# Dodge Mike's Newport Send — Launch Checklist

## Highest-impact polish

- Add 5 gameplay screenshots to the repo and App Store listing.
- Add one short gameplay GIF or 15–30 second trailer.
- Confirm the soundtrack is cleared for public release and App Store use.
- Make sure README gameplay details match the live game exactly.
- Test on iPhone Safari, Android Chrome, desktop Chrome, and desktop Safari.
- Test GitHub Pages after every major change.

## GitHub repo polish

- Add screenshots under `/screenshots`.
- Add a GIF named `gameplay.gif` if possible.
- Update README with a visual hero section.
- Add clear Play Now link near the top.
- Keep `PRIVACY_POLICY.md`, `TERMS.md`, and `SUPPORT.md` in the repo root.
- Add a short repo description in GitHub settings:
  `A neon Newport Beach arcade dodger where Mike survives falling bar signs, tilt controls, and 10 levels of Send energy.`

## Game polish

- Add a How to Play modal or clearer intro copy.
- Make the mobile settings button easy to tap.
- Keep score, best score, level, and time visible on all screen sizes.
- Confirm tilt controls work after motion permission.
- Confirm arrows/WASD work on desktop.
- Confirm Share Link works on mobile.
- Confirm local score saving works after refresh.
- Confirm audio starts only after Play.

## App Store prep

- App icon: 1024 x 1024 PNG.
- Screenshots: required device sizes for iPhone.
- App preview video if available.
- Privacy Policy URL.
- Support URL.
- Marketing URL, optional.
- App subtitle.
- App description.
- Keywords.
- Category: Games / Casual.
- Age rating questionnaire.

## Website/PWA prep

- Confirm `manifest.webmanifest` loads with no console errors.
- Confirm app icons load correctly.
- Confirm Apple touch icon loads on iPhone home screen.
- Confirm theme color looks good.
- Confirm the game can be added to home screen.

## Pre-launch QA script

1. Open the game fresh.
2. Press Play.
3. Confirm music starts.
4. Move with keyboard.
5. Open Settings.
6. Change sensitivity.
7. Save settings.
8. Play until Game Over.
9. Save initials.
10. Refresh page.
11. Confirm best score persists.
12. Try Share Link.
13. Repeat on mobile with tilt.
