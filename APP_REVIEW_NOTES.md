# App Review Notes

## Overview

Dodge Mike's Newport Send is a casual neon arcade game inspired by Newport Beach.

The player controls Mike through 10 themed districts, collects power-ups, builds combos, and tries to finish the full run with the highest score possible.

## Gameplay

- 10 levels
- 30 seconds per level
- 3 lives per level
- Lives reset at the start of each new level
- Score carries across the full run
- Local Top-5 leaderboard
- No account required

## Data and accounts

The current version does not require an account.

High scores, initials, mute settings, and best-run stats are stored locally on the player's device or browser.

The current version does not use an online leaderboard.

## Monetization

The current version does not include in-app purchases.

The current version does not include ads.
