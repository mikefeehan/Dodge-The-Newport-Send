# Support

Need help with Dodge Mike's Newport Send?

Email: support@bet2sweat.com

Please include:

- Device type
- Browser or app version
- What happened
- Steps to reproduce the issue
- Screenshot or screen recording if available

Common fixes:

- Press Play before expecting music to start. Browsers usually require a tap before audio can play.
- On iPhone, allow motion access when prompted if using tilt controls.
- Use arrows or WASD on desktop.
- If the game feels too sensitive, open Settings and adjust sensitivity.
- If saved scores look wrong, clear browser storage and restart the game.
